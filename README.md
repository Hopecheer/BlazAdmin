# BlazAdmin

#### 介绍
基于Blazui的后台管理模板，无JS，无TS，非 Silverlight，非 WebForm，开箱即用

#### 使用前提
参考Blazor使用的前提条件

1. 安装 .Net Core 3.0
2. 安装 VS2019
3. 安装所有 VS2019 Blazor Extension


#### 计划特性

1.  一个标签，进行一些设置，得到一个后台管理界面 :fa-spinner: 
2.  标签页式设计，天然模块化设计
3.  集成JWT认证、用户登录
